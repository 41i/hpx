Testing HPX

To ensure correcteness of HPX, we ship a large variety of unit and regression tests. The tests are driven by the CTest tool and ran automatically by buildbot on each commit. In addition, it is encouraged to run the test suite manually to ensure proper operation on your target system. If a test fails for your platform, we highly recommend to submit an issue on our bug tracker with detailed information about the target system.
